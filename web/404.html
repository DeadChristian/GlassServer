FAQ

How do I install Glass?

Download Glass.exe and run it. No installer needed. If SmartScreen warns you, click More info → Run anyway.

How do I unlock Pro?

Buy once on Gumroad. In the app, open Pro → Enter license…, paste your key, and activate. Pro unlocks instantly.

Do I need internet every time?

After one successful validation, Glass grants a 7-day offline grace period. When you’re back online, it refreshes automatically.

What’s Free vs Pro?
  • Free: Opacity + live apply, save per-window, theme, and tinting the Glass app itself (doesn’t count).
  • Pro: Always-on-top, Click-through (toggle + hold-key), Lock, Quick preset, Globe, Theme Designer.

Shortcuts

  • Ctrl + Enter — Apply
  • Ctrl + 0 — Reset to 100%
  • Ctrl + ↑/↓ — Nudge opacity ±5%
  • Ctrl + T — Always on top (Pro)
  • Ctrl + G — Click-through (Pro)
  • Ctrl + L — Lock (Top + Click-through) (Pro)
  • Hold Left Alt — Momentary pass-through (Pro)
  • F5 — Refresh window list

Settings and license live at %APPDATA%\Glass\. We do not collect usage telemetry.

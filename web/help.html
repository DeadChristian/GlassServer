Glass - Help Guide

Glass is a lightweight Windows utility that allows you to control window transparency with ease.

Features

  • Auto-refreshing window list
  • Set opacity of any window
  • Pin windows to the background
  • Focus Mode: fade all non-target windows
  • Customizable themes
  • Memory for window states
  • System tray support
  • Offline use only — secure by design

Hotkeys

HotkeyAction
Ctrl + Alt + ↑Set app to 100% opacity
Ctrl + Alt + ↓Set app to 50% opacity
Ctrl + Alt + HHide Glass window
Ctrl + Alt + SShow Glass window
Ctrl + Alt + TApply transparency to selected target
Ctrl + Alt + BPin or unpin the target window
Ctrl + Alt + FFocus mode (fade other windows)
